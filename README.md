## 软件介绍 🌊

这是一款能让您在教室里玩失踪的神器！(￣▽￣)~* 无论是老师突然来访，还是女朋友查岗，「潜水艇控制面板」都能帮您一键"消失"！(●'◡'●)（开玩笑的啦，请合理使用哦 ╮(╯▽╰)╭）

---

## 核心特性 🚀

- **自动更新**：始终保持最新版本！
- **公告系统**：开机显示最新动态。
- **隐藏窗口功能**：一键消失，想回就回。
- **高度自定义**：快捷键 & CMD 指令自由配置。
- **用户友好界面**：简洁直观，零门槛上手。

---

## 基础操作 📘

### **1. 启动潜水艇 🚀**

- 双击运行，欢迎来到控制面板！
- **第一次启动需要管理员权限**，别害羞，点 "是" 就好啦 (｡◕‿◕｡)

### **2. 添加新艇 ➕**

- 选择想要“藏起来”的程序。
- 设置快捷键（建议 `Alt+D` 之类的组合）。
- 支持多程序，随心隐藏！

### **3. 紧急潜航 🎯**

- 按下快捷键，程序瞬间“蒸发” ∑(￣□￣;)
- **再按一次，程序复现**  (〃'▽'〃)
- 声音也会自动静音，超级贴心！

### **4. 托盘模式 🖥️**

- 关闭窗口时，潜艇会潜入系统托盘。
- 右键托盘图标可快速召唤菜单！

---

## 默认快捷键 ⌨️

| 快捷键   | 功能          |
| ----- | ----------- |
| Alt+H | 隐藏/显示托盘图标   |
| Alt+T | 隐藏/显示任务栏    |
| Alt+N | 断开/连接网络     |
| Alt+P | 窗口置顶/取消置顶   |
| Alt+C | 隐藏/显示光标位置程序 |
